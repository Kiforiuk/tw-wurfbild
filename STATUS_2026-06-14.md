## 🔄 NÄCHSTES ZIEL: Progressive Web App (PWA)

### Was ist PWA?
- App wird **installierbar** (wie native App)
- Funktioniert **offline** nach einmaligem Laden
- Trainer kann **im Spiel ohne Internet** arbeiten
- Perfekt für Spielfeld-Nutzung

### Implementierungs-Plan (45 Minuten)
1. **Service Worker** (`public/sw.js`) - Offline-Caching
2. **Web App Manifest** (`public/manifest.json`) - App-Metadaten
3. **"Install App" Button** - Im Header hinzufügen
4. **HTML-Link** - Im index.html verlinken
5. **Testing** - Lokal testen & auf Website deployen

### Offline-Capabilities nach PWA-Implementierung
- ✅ App laden & Daten erfassen
- ✅ Statistiken anschauen
- ✅ Würfe speichern (LocalStorage)
- ⚠️ Import/Export braucht Internet (externe Datei-Quellen)

## 📝 TECHNISCHE DETAILS

### Tech-Stack
- React 18 + Vite
- Tailwind CSS (inline styles für Absolut-Positionierung)
- XLSX Library (Import/Export)
- Lucide React (Icons)
- LocalStorage (Persistierung)

### Daten-Struktur
```javascript
{
  id: 1718265600001,
  time: "00:15",
  torwart: "TW 1|2|3",
  gegenspieler: 7,
  wurfposition: "LA|RL|RM|RR|RA|KM|7m|TG",
  macroZone: 1-9,
  microZone: 1-9,
  ergebnis: "tor|gehalten|vorbei"
}
```
